# NAS SYNC

## Requirements

* LFTP : [https://lftp.yar.ru/](https://lftp.yar.ru/)

## Installation

* Clone this repos

```bash
$ git clone https://github.com/qmachard/nas-sync.git
$ cd ./nas-sync
```

* Copy and edit the configuration file

```bash
$ cp config/config.sample.ini config.ini
```

* Run !

```bash
$ ./sync-movies.sh
```
